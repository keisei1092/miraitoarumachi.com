About

1993年11月3日生まれ、茨城県かすみがうら市出身。
東京電機大学 未来科学部 情報メディア学科を2016年4月に卒業予定。
2歳の頃から父親の膝の上でPowerPC (Macintosh) を使ってゲームをしたことが始まりで、以来小学生時代にWeb 2.0の文化と対峙、中学・高校時代にYouTubeとニコニコ動画を中心とするCGMやソーシャル・ネットワーキングの文化に触れて生きてきた。
大学では即決で情報系学域へ、苦手な数学や物理系をはじめ危なげない受講態度になりながらも何とか3年生ではフル単まで持ち直し無事卒業予定。
研究室ではスマートフォンや環境センサによりユーザの生活を支援し、安心安全を保つ実世界環境の構築という目標のもとで、人間・防災・都市などをテーマにした研究を行っている。卒研テーマは現在熟考中。

部活動は、パソコン部が無かったため中学・高校と吹奏楽部に入りフルート（ピッコロ）を練習してきた。中高と通して成績は県大会で銀賞程度のレベルであったが、自分自身は吹奏楽が好きになり熱心に取り組み、その過程で一定のレベルの相対音感を取得、打楽器の音階が分かるまでになった。なお高校ではもともとバンドがやりたかったため軽音楽部でベースも弾いていた。
先述の部活動での音楽知識を生かし、ニコニコ動画で衝撃を受けたVOCALOIDでの楽曲の制作を2009年に開始。街や都市が好きだったため、「ミライトアルマチ構想」という名の個人サークルを創設し以来ニコニコ動画で1万再生を1つ達成し、全動画の総再生数としても10万に肉薄する勢いを得ている。同人でのCDの企画及び発売経験が2度あり、最新CDは手売りで当日100枚を売り、現在も通販で1月5枚程度ずつ売れている。

創作活動を宣伝するため、そして様々なソーシャル・ネットワーキングサービスに散らばった自分のアカウントを繋げるポータルサイトのようなものを作るという名目で、小学3年生の頃FFFTPの使い方も分からず父親に泣きつきながらWebサイトの制作を開始。現在は、HTML5・CSS3・JavaScriptの勉強を中心に大学生活の傍らで制作を続けている。
最近は、MySQLとPHP・Java・Rubyを使ったWebサービスの開発を嗜むべくVPSサーバーでのサイト運用を開始。Linuxの学習、Webサーバー、そしてセキュリティの問題など、学ぶことは多いが楽しんで練習している。

これからは、Webエンジニアとしてのキャリアを積み、その過程で未来を見る眼を養い、沢山の人を幸せにできる一大事業を起こすか、その開発に携わりたい。IT技術を信じ科学は人を滅ぼすわけではないことを証明するべく精一杯働く覚悟と決意に燃えている。

らぼログ ランキングとレビューで分かる研究室

大学の講義「サーバプログラミング演習」で同期5人で結成したチームで開発したWebサービス（公開予定は無し）。要件定義、設計、実装という一連のプロセスを初めて経験した。自分はチームの中ではアプリケーションスペシャリストとして、画面遷移図、ユースケースの制作とデータモデルなど、そして全てのフロントエンド実装を担当した。
サービスの内容としては「食べログ」のテーマを大学の研究室配属でそのままそっくり置き換えたようなサービスである。大学にある研究室に研究内容や研究室の居心地などの評価ポイントを設け、ユーザに自由にコメントをしてもらい、ランク付けするようなものとなっている。
単位のため講義のために制作したサービスであり実際に動作させてはいないが、チームメイトが作った取扱説明書など、資源は沢山残っているので、もし興味を持って頂けたら是非お見せする次第である。

  • Tool: Java, Apache Tomcat, PostgreSQL, HTML, CSS, JavaScript
  • Date: 2014.4 - 2014.7
  • Type: Web Development

協調フィルタリングによるVOCALOID動画推薦システム

大学の講義「Web情報システム演習」で1人で制作したシステム。個人では把握し切れない量に膨張したニコニコ動画のコンテンツを、Twitterでの嗜好が似たファンのツイートをクロールすることにより推薦できないかという思いで、協調フィルタリングシステムを制作した。つぶやきを参照しユーザがマイリストをしていた場合やポジティブなコメントを添えていた場合に重み付けする点がPRポイント。詳しくは 協調フィルタリングによるVOCALOID動画推薦システム - SlideShare を参照されたい。
将来展望としてはもちろんWebアプリ化まで達成したいと思っている。

  • Tool: Java, MySQL, Apache Commons HttpClient, Apache Common Math Library, Niconico Douga Search API, Twitter REST API, JSON
  • Date: 2014.11 - 2015.1
  • Type: Web Development

DepartureBoard

電車が好きで特に駅の電光掲示板に旅情を感じるため、パソコンのウィジェットにしたいと思い1人で制作した。機能は現在時刻に対する次の電車とその次の電車の時刻・行先・発車番線などを表示し、デザインはJR東日本の発車標に極力のっとっている。時刻表データは同じhtmlファイルの中に配列として直接手打ちした。
将来展望としては、時刻表データを自前で入力できるようにしたり、えきから時刻表のようなサイトから拾ってこれるように、そしてデザインもJR東日本以外にも東京メトロや私鉄のデザインも制作し、全国の鉄道ファンのニーズを満たせるものにしていきたい。
実物は ここ にある。

  • Tool: HTML, CSS, JavaScript
  • Date: 2014.2 - 2014.3
  • Type: Web Development

iBeaconを用いたおかえりアプリ

研究室での学園祭の発表のために作ったiPhoneアプリ。趣旨としては家にbeaconを置いておき、アプリを起動した状態で家に帰るとbeaconに反応してアプリ内のキャラクターが挨拶してくれる。詳しくは こちら 。beaconを家中に置いておき、キッチンやお風呂、さらには寝室などで独自の反応をさせられる、挨拶アプリを目指している。モデルも変えられるようにすれば自分の好きなキャラクターとコミュニケーションができる。

  • Tool: iBeacon (Bluetooth Low Energy), Objective-C
  • Date: 2014.10
  • Type: Sensing

Web Design

自分の創作活動を宣伝したい名目で、昔からWebサイト制作を行ってきた。その中で、「魅せる」ことを意識し始めるようになり、デザインにも力を入れるようになった。
最近はVPSサーバーを借り、CentOSとApache Webサーバーを用いたサイト制作に転換、バックエンドとセキュリティにも興味範囲を広め鋭意勉強＆活動中。自分のWebサイトは現在最小限のデザインになっているため成果物としては掲載しかねるが、大学内で案件で幾つかWebデザインを手伝ったためそちらを成果物として掲載する。
・母校学園祭のWebサイト のデザインを担当。
・母校で所属しているサークルの公式Webサイト のデザインを担当。
・母校で所属しているサークルのWebサイト（２） のデザイン。

  • Tool: HTML, CSS, JavaScript, GIMP, InkScape
  • Date: since 2003
  • Type: Web Design

TrackReferrer

Web Audio APIに興味を持ち、やってみたいと思ったことを体当たりで実装してみたアプリ。楽曲をトラックごとに切り出したmp3ファイルを一度に再生し、個々のトラックをミュートすることができる。楽曲の構造を見える化し、楽曲制作に興味のある人への導入になるようなアプリになると考えている。

  • Tool: Web Audio API, HTML5, JavaScript
  • Date: 2014.4
  • Type: Web Development

CREATOR ACTIVITY

ニコニコ動画でのVOCALOID界隈のCGM文化に衝撃を受け、2009年からVOCALOID「初音ミク」を用いた創作活動を開始。ニコニコ動画とYouTubeでの総再生回数は10万を超える。ソニーモバイルコミュニケーションズのスマートフォン「Xperia feat. HATSUNE MIKU」の公式アプリケーション「Find Vocalo-P Player」にて創作活動が紹介され、書き下ろし着信音の配信経験がある。2013年、2014年に同人アルバムを頒布、過去200枚以上を売り上げる。海外のキュレーター及びクリエーターやファンとの交流も積極的に行い、現在カリフォルニア、ニューヨーク、ローマなどのクリエーターと交流を継続、東南アジアや南米からのファンも多い。
ニコニコ動画のマイリスト, YouTube Channel

  • Tool: Logic Pro X, VOCALOID 2 Hatsune Miku
  • Date: 2009.10 - Now
  • Type: Creator Activity
